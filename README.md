# image_processing
image pocessing using bilinear intepolation
this project is basically about the image resizing using hadware using bilinear interpolation in which precision is fixed upto 8 bits(mentioned in the problem statement) and this project also include the gaussian blur logic to improve the ssim quality while down scaling.This project does both upscaling and down scaling on the image.
The poject basically recieves the hex file from the python code(which convert image to hex file) and the code basically upscale and downscale based on the parameter set by the user.
